Predicts wins/standings for the upcoming 2025 NFL season using 2 algorithms (1 linear regression and 1 random weighted model) which take training data from 2022-2024 preseason stats/rankings on Vegas win lines, QB rankings, Coach rankings, OLine rankings, D rankings, and SOS. These are compared to those years actual outcomes to find the best model to predict the outcome for the 2025 seasons given the same preseason data.
